# HCI-Handbook
R source code and CSV data for the following HCI Handbook (4th ed.) chapter:

Wobbrock, J.O. (2026). Experiment design and statistical analysis in human-computer interaction. Chapter 29 in J.A. Jacko (ed.), Human-Computer Interaction Handbook (4th ed.). Boca Raton, Florida: CRC Press, pp. 779-846.

Details:

This Github repository contains the R source code files and CSV data files for Chapter 29 in the HCI Handbook (4th ed., CRC Press, 2026) written by Jacob O. Wobbrock. The R files appear in the R directory and the CSV files in the data/ subdirectory. The R code is written to read in the CSV files assuming they are in a data/ subdirectory. The R source code reads in the CSV files and performs various statistical analyses on them as described in the chapter. The HCI Handbook chapter contains excerpts from these R source code files and their output. The reader is strongly encouraged to open the R source code files and step through them while reading the HCI Handbook chapter.

Please send any inquiries or errata to Jacob O. Wobbrock at wobbrock@uw.edu.
